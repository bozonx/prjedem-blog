# Blog of project Edem

## Run

Use node version: 18 or greater

- Install dependencies:

  ```bash
  yarn
  ```

- Run locally:

  ```bash
  yarn dev
  ```

- Admin build and preview:

  ```bash
  yarn admin
  yarn build && yarn preview
  ```
